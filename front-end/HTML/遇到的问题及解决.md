## 3 input 
### 3.1 radio
#### 3.1.1 禁用radio
`disabled="true" `
#### 3.1.2 radio与label绑定
    for 属性规定 label 与哪个表单元素绑定。 
    点击“是”文字时，相当于点击id为isSealed1的radio。
```
<input id="isSealed1" type="radio"/><label for="isSealed1">是</label>
<input id="isSealed0" type="radio"/><label for="isSealed0">否</label>
```
### 3.2 span
